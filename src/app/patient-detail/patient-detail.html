Configurar Origen Seguro

Para usar la File System Access API desde HTTP, necesitas configurar tu navegador para tratar este origen como seguro.

¿Qué hace este archivo?

Configura Chrome y Edge para tratar {{ currentOrigin }} como un origen seguro, habilitando todas las APIs que normalmente requieren HTTPS.

Pasos a seguir:

  1. Descarga el archivo de configuración (.bat)
  2. Ejecuta el archivo como administrador (clic derecho → "Ejecutar como administrador")
  3. Cierra TODOS los navegadores (Chrome/Edge)
  4. Abre nuevamente el navegador y recarga esta página
  5. La funcionalidad de archivos debería estar disponible

¿Es seguro?

Sí, esta configuración solo afecta a este origen específico y es la solución recomendada por Google para aplicaciones empresariales.
